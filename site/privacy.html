Privacy

The website is static. The command-line application keeps its ledger on your machine.

This website

The site has no analytics, advertising, cookies, forms, accounts, or client-side JavaScript. It does not receive or upload your Motus ledger.

The hosting provider may process standard web request data such as an IP address, requested URL, browser information, and request time to deliver and protect the site. See GitHub's privacy statement for its practices.

The command-line application

Motus stores selected command metadata in .motus/ledger.db at the current Git root or working directory unless you choose another state path. It has no telemetry or ledger network client.

Command argument values, command stdin, raw stdout and stderr, environment variables, source files, prompts, and transcripts are not stored.

Finding text and closure notes are stored only when you submit them through a file or standard input. They remain in the local ledger until you delete the ledger file.

Your control

The ledger is an ordinary local SQLite file. You choose where it lives, whether it is backed up, who can access it, and when it is deleted. Add .motus/ to the project's ignore file if it should remain outside version control.

Motus Work Ledger is open source under Apache-2.0.
